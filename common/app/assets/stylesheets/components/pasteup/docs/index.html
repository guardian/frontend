Pasteup is where design meets development. It is where the Guardian’s globally recognised design language is turned into code for the web, and the starting point when styling GNM branded products for both internal teams and third parties.

Status

Pasteup is currently in development. The base CSS and layout CSS can be used and are powering the documentation pages you are currently reading.

You can see what modules have been implemented in the module library.

Documentation

Pasteup is a library of code to help you implement the Guardian’s design and interaction patterns on the web. It can be used in various ways, most simply by linking directly to a CDN hosted version, or if necessary pulling individual modules into your own project’s build process.

Base CSS

The base CSS of Pasteup should give you a generic Guardian style design for your page. This includes standard fonts and font sizes, colors, and heading styles. This is the simplest way of using Pasteup, and should go some distance to giving you a Guardian look and feel.

You can see an example of how this might look in our styleguide.

Layout styles

grid.html

Guardian web pages typically implement one of two grid systems. A 940px fixed width grid as on guardian.co.uk, and a liquid and responsive layout which works across devices from mobile and tablet up to desktop (this doesn't exist yet).

Module library

modules.html

A set of stand-alone components which make up Guardian web pages. These should be entirely decoupled from any particular page or any particular layout.

Hosting

Pasteup is fully hosted on http://pasteup.guim.co.uk which is a globally distributed and load balanced CDN. All files are minified, gzipped, and have far future Expires headers so they are aggressively cached by browsers.

In the first instance, please link to URLs on this domain, as they are likely to be already cached by your users.

Getting started

The quickest way to start is to add a link to the main Pasteup CSS file in the <head> of your document. This will style your document with some standard Guardian typography and colours.

Hosting

Pasteup is available on http://pasteup.guim.co.uk which is hosted on a globally distributed and load balanced CDN. All files are minified, gzipped, and are served with far future Expires headers so they are aggressively cached by browsers.

Typically you should link to URLs on this domain, as they are likely to be already cached by users.

Current production URLs

  • http://pasteup.guim.co.uk/<%= pasteupVersion %>/css/pasteup.min.css
  • http://pasteup.guim.co.uk/<%= pasteupVersion %>/js/require.js
  • http://pasteup.guim.co.uk/<%= pasteupVersion %>/js/main.min.js

Build

Pasteup is built using LESS and the RequireJS optimizer. If you are using these tools in your project, it should be simple to include specific modules of Pasteup, and integrate them into your build.
