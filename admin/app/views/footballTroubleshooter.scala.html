Football Troubleshooter

How to access the raw feeds to determine if a problem lies in the feeds or in the application

The current API host is @pa.footballHost and the API key is @pa.footballApiKey.

To log a support call with PA email customer.services@@pressassociation.com or call 01430 455337

If in doubt raise a support ticket. (it's what we pay them for)

The PA API explorer is here.

Matchday Feed

This is the feed you are most likely to be interested in. This feed shows detailed information on today's matches and is the feed we use while the match is live.

A single call to this api loads match data for today for all competitions and the date format is YYYYMMDD.

This api call also includes today's fixtures and results.

Here is the API call for today's live matches:

@pa.footballHost/api/football/competitions/matchDay/@pa.footballApiKey/@today

Competitions feed

This shows us all the competitions we have access to. You can use it to find the ID of a competition that we can use to access other feeds.

@pa.footballHost/api/football/competitions/competitions/@pa.footballApiKey

Fixtures feed

Matches will first appear in the fixtures feed. We query each competition separately for all upcoming fixtures.

Here is the API call for all upcoming fixtures in the @competition (id @competitionId). You can get other competition IDs from the Competitions feed above.

@pa.footballHost/api/football/competition/fixtures/@pa.footballApiKey/@competitionId

Results feed

Shows the results for completed matches. We query each competition separately for results.

Here is the API call for results in the @competition (id @competitionId). You can get other competition IDs from the Competitions feed above and the date format is YYYYMMDD.

@pa.footballHost/api/football/competition/results/@pa.footballApiKey/@competitionId/@Format(new LocalDate(2014, 6, 12), "yyyyMMdd")

League table

Shows all the stats we use to populate the league tables.

Here is the API call for the league table for @competition (id @competitionId). You can get other competition IDs from the Competitions feed above and the date format is YYYYMMDD.

@pa.footballHost/api/football/competition/leagueTable/@pa.footballApiKey/@competitionId/@today

Match stats

Detailed match stats for a specific match. You would get the match ID from either the fixtures or matchday or results feed.

@pa.footballHost/api/football/match/stats/@pa.footballApiKey/3630511

Line ups

Team line ups for a specific match, containing team colours (before we apply darkening, and missing colours default to #eeeeee).

@pa.footballHost/api/football/match/lineUps/@pa.footballApiKey/3630511 } }
