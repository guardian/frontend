Rules:

This slot will by default accommodate an ad of height 250px on business fronts unless there is an ad that:

  • Has the CPD cost type
  • Targets the responsive size (88x70) in which case the slot will be responsive height by default
  • Targets the leaderboard size (728x90) in which case the slot will be leaderboard height by default
  • Targets the business front ad unit directly, either through line-item targeting or creative targeting
  • Geotargets the countries UK, US or Australia

The ad will also appear on the Code site if it targets a test cookie.

@commercial.fragments.slot(slotReport) }
